# DSA210

Project Proposal: DSA 210 proposal 
1. Motivation
The reason I chose this project is because I wanted to have an idea of how many transactions at IstanbulKart online system, what type of transaction and what is the frequency of these transactions. I will analyze the data and apply data science methodologies to solve this problem. 
2. Data Source
I have collected the data from the IBB online free data base. It was a public data base, and you can chose which ever you want. My aim was to find out how many transactions were made during the year 2024 and type of the transactions.

3.Motivation: 
My motivation by choosing this data is to understand how many transactions happened during the year 2024. I wanted to make research out of this because of the waiting line at the Istanbul Kart refill machines and the number of errors that I have faced while uploading money on my card from online. I noticed that most of the transactions happens during the winter season. I found out that this was the only time I had problem with system so I chose this data as my project.
